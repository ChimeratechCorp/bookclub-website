USING MATH TO DRAW A WHEEL !!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!
360 degrees is 2π radians
If we have 3 books, each slice is 120 degrees, which is 2π/3 radians
If we have 4 books, each slice is 90 degrees, which is 2π/4 radians
If we have 5 books, each slice is 72 degrees, which is 2π/5 radians
So the formula for slice size is 2π / n
n = number of books in arrary!
